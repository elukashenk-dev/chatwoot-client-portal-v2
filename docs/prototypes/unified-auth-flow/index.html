Auth Flow Prototype

Единый вход по коду

Прототип убирает отдельную регистрацию. Пользователь вводит email, подтверждает код, принимает документы только при первом доступе и попадает в чат.

Это design artifact: без API и без записи данных. Тексты и порядок экранов здесь можно править до переноса в production-компоненты.

Все экраны сценария

Сетка нужна для быстрого ревью плотности, текстов, маргинов и логики переходов.
